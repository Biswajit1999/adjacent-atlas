# Security Policy

## Supported versions

Adjacent Atlas is in early development. Security fixes are applied to the
latest `main` and the most recent tagged release.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Report vulnerabilities privately through GitHub's
[private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
on this repository (Security tab → Report a vulnerability).

When reporting, include:

- a description of the issue and its impact,
- steps to reproduce,
- affected versions or commit, and
- any suggested mitigation.

You can expect an acknowledgement within 7 days. Once a fix is available, we
will coordinate a disclosure timeline with you before publishing details.
