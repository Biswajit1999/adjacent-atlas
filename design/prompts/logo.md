# Logo — generation prompt

A wordmark plus a single geometric mark. The brief below is for an image or
vector tool; the current mark in the app is a CSS square, intentionally simple.

## Mark

> A small, precise mark suggesting a node in a constellation: a bright rounded
> square or point with a faint concentric halo, as if seen through an
> instrument. Single accent colour (#5eead4) on near-black (#0b0f14). No
> gradients beyond a soft glow. Flat, geometric, calm. Should read at 16px.

## Wordmark

> "Adjacent Atlas" set in Inter, 600 weight, tight tracking. Lowercase descender
> alignment with the mark. The mark sits to the left of the text with ~10px gap.

## Don'ts

- No globe or planet cliché.
- No 3D bevels, no drop shadows beyond a faint glow.
- No more than one accent colour.
