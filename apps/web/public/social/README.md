# public/social

Static social and screenshot assets served from `/social/...`.

## Contents

- `card.svg` — generated by `pnpm.cmd social-card` from the current snapshot
  (real ranked nodes). Regenerate it whenever you build a new snapshot. It is not
  committed by default; commit it deliberately if you want it served.
- Screenshots — optional, captured locally per `docs/screenshots.md` and
  `design/prompts/demo-screenshot.md`. Reference them from the README with
  relative paths.

## Notes

- No stock or fabricated images. Every asset here should come from the app or the
  documented generators.
- The app also exposes a dynamic Open Graph image at `/opengraph-image`
  (`apps/web/app/opengraph-image.tsx`); you usually do not need a separate static
  OG file unless a platform requires one.
